# Project Ethos

Systray Wrap Doubler is a small tool, but it follows a larger rule:

Build useful things for real people.

## Principles

- Keep the product truth clear.
- Do not hide tradeoffs.
- Do not use fake urgency.
- Do not sell confusion.
- Do not make uninstall or recovery difficult.
- Keep source and documentation available so the work can be inspected, fixed, forked, or carried forward.

## Monetization

Charging for work is allowed.

Extraction is not.

If this project ever grows beyond a free utility, the pricing and ownership terms should be plain enough that a normal home user can understand:

- what they get
- what it costs
- what can stop working
- what remains theirs
- how to leave cleanly

## Handoff

If someone wants to improve this project, adapt it, or turn the approach into a Windhawk mod or another open-source form, take the torch.

Keep the map visible:

- source code
- documentation
- public changelog
- revert path
- uninstall path
- honest limits

This project was made by Al with help from ChatGPT and Codex. The point is not ownership-as-a-wall. The point is proof that the thing can be built, shared, understood, and improved.
